# Social Media Platform Specs

This reference is used by the `social-media-creator` skill.

## LinkedIn

| Property     | Spec                                           |
| ------------ | ---------------------------------------------- |
| Post length  | Max 3,000 chars; **ideal 150–300 words**       |
| Native video | Up to 10 min; captions strongly recommended    |
| Image        | 1200 × 628 px for link previews                |
| Hashtags     | 3–5 (more hurts reach); place at end           |
| Mentions     | `@FirstName LastName` for individuals          |
| Best time    | Tue–Thu, 8–10am and 12–2pm (audience timezone) |

### LinkedIn Post Formula

```
[Hook — bold claim or surprising stat]

[2–3 short paragraphs expanding on the idea]

[Takeaway or lesson in 1 sentence]

[CTA — question or "Follow for more…"]

#Hashtag1 #Hashtag2 #Hashtag3
```

### LinkedIn Do / Don't

| Do                                      | Don't                                 |
| --------------------------------------- | ------------------------------------- |
| Start with a hook that stops the scroll | Start with "I'm excited to announce…" |
| Use line breaks liberally               | Write dense paragraphs                |
| Share a specific experience or number   | Stay vague ("I learned so much")      |
| Engage with early comments (first hour) | Post and disappear                    |

## X (Twitter)

| Property      | Spec                                    |
| ------------- | --------------------------------------- |
| Single tweet  | Max **280 chars**                       |
| Thread tweets | Each ≤ 280 chars; number them (1/n)     |
| Image         | 1200 × 675 px (16:9)                    |
| Hashtags      | 1–2 max; embedded in sentence preferred |
| Best time     | Mon–Fri, 8am–12pm                       |

### X Post Formulas

**Bold claim:**

```
[Provocative statement in ≤ 240 chars]

[Optional: link or image]
```

**Thread opener:**

```
The [thing] nobody talks about:

🧵 (1/7)
```

**Observation:**

```
Hot take: [opinion]

Here's why ↓
```

## Instagram

| Property       | Spec                                                            |
| -------------- | --------------------------------------------------------------- |
| Caption length | Max 2,200 chars; **ideal first 125 chars carry the hook**       |
| Hashtags       | 5–15; mix of niche and broad; can go in first comment           |
| Image ratio    | 1:1 (square) or 4:5 (portrait) for feed; 9:16 for Reels/Stories |
| Best time      | Mon–Fri, 11am–1pm and 7–9pm                                     |

### Instagram Caption Formula

```
[Relatable hook or question — visible before "more"]

[Story or value in 2–4 short paragraphs]

[CTA — save, comment, share, or tag someone]

.
.
.
#hashtag1 #hashtag2 #hashtag3
```

### Instagram Do / Don't

| Do                                          | Don't                              |
| ------------------------------------------- | ---------------------------------- |
| Use emoji to break up text                  | Overload with emoji                |
| Lead with a hook before the "more" fold     | Bury the point                     |
| Use 1–3 highly relevant hashtags in caption | Dump 30 hashtags in the caption    |
| Ask a question to drive comments            | Use a weak CTA like "Check it out" |

## Cross-Platform Repurposing

| Source → Target       | Key Adaptation                                 |
| --------------------- | ---------------------------------------------- |
| Long blog → LinkedIn  | Extract the top 3 insights; write as list post |
| LinkedIn → X          | Take the hook sentence; cut to ≤240 chars      |
| LinkedIn → Instagram  | Rewrite conversationally; make it visual-first |
| Webinar → Thread      | Each key point = one tweet                     |
| Case study → LinkedIn | Lead with the result; build back to the story  |
